Intrucciones para Tarea 3:

-Abrir servidor en algun host conectado a la Red. 

-Se pueden conectar hasta 5 clientes a la vez.

-Se manejan la entrada de clientes como sesiones.

-Cuando entra un cliente se le asigna un nombre de usuario en la red. (Ej: u_2)

-Hay una carpeta que se comparte en la red, esta se llama 'sharing' y estara en el directorio del programa.

-Los archivos dentro de esta carpeta seran ofrecidos a otros usuarios y vice versa.

-Cada cliente quedara en un thread distinto en el servidor, esperando que le lleguen comandos.

-El cliente le ofrecera al usuario opciones de uso:

	1- File List: Muestra una lista de archivos disponibles para ser descargados de la red ordenados por usuario.

	2- Download: Te permite elegir un usuario y te conecta con él, luego te permite descargar uno de sus archivos compartidos, esto se logra en 2 pasos
		1. Escribir el nombre de usuario del cliente deseado.
		2. Escribir el nombre del archivo que se desea descagar.
